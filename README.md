# NSW Opportunity Class (OC) School Data

Open dataset of all NSW Opportunity Class schools — placements, NAPLAN rankings, and tier classifications.

## What is the OC Test?

The Opportunity Class (OC) Placement Test is a competitive exam for Year 4 students in New South Wales, Australia. Successful students gain entry to dedicated classes in 88 public schools for Years 5 and 6. The test covers Reading, Mathematical Reasoning, and Thinking Skills.

**2026 Test Date:** May 8, 2026

## Dataset Contents

| File | Description |
|------|-------------|
| `schools.csv` | All 88 OC schools with addresses, regions, placement numbers, and NAPLAN rankings |
| `tiers.csv` | School tier classifications (S/A/B/C) based on 2024 NAPLAN data |
| `competition_stats.csv` | Key statistics about OC competition |

## Key Statistics

- **88 schools** across NSW (57 metro + 31 regional)
- **~1,840 places** available each year (includes Aurora College virtual class with 100 places)
- **~4,000+ applicants** competing annually
- Places range from **15 to 60** per school

## Tier System

Schools are classified into tiers based on 2024 NAPLAN performance among OC schools:

| Tier | Rank Range | Schools | Description |
|------|-----------|---------|-------------|
| S | #1–10 | 10 | Top-performing OC schools |
| A | #11–25 | 15 | Strong academic results |
| B | #26–50 | 25 | Solid performance |
| C | #51–87 | 37 | All other OC schools |

## Data Sources

- [NSW Department of Education](https://education.nsw.gov.au) — Official school list and placement policies
- [ACARA My School](https://myschool.edu.au) — NAPLAN performance data
- Third-party aggregators for cross-referenced rankings

## Important Notes

- Rankings are based on **2024 NAPLAN data** and may change annually
- Mid-range rankings (#25–60) show significant year-to-year variation
- Top 20 rankings are relatively stable across data sources
- This dataset is for informational purposes — always verify with official sources
- Glenbrook PS is listed but not available for 2027 placements
- Blaxland East PS shows special notation in official sources
